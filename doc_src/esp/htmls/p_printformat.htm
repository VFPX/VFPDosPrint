PrintFormat (Propiedad)

Permite indicar el nombre del archivo de formato a utilizar para el informe. El uso de los archivos de formato en conjunto con el método Run() constituyen la forma más fácil de generar informes de texto usando la clase DOSPrint.

Ejemplo(s):

*-- Se genera el informe básico de clientes
*
#DEFOBJ oRep AS DOSPrint
oRep.PrintFormat:="INFBASCLI.FMT"
oRep.Run("QCLIENTES")
oRep.Print()
#UNDEFOBJ oRep
