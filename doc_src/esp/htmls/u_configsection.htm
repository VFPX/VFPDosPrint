Sección CONFIG

La sección CONFIG se utiliza para configurar las características del informe tales como el tamaño de la página, los márgenes, etc. Cada linea dentro de la sección corresponde a un par Propiedad=Valor, donde Propiedad corresponde a cualquiera de las propiedades disponibles para la clase DOSPrint. Si una propiedad no es indicada en la sección CONFIG, se asume su valor por defecto.

Las propiedades lógicas pueden ser configuradas usando las constantes Si, True o Yes para activarlas y No o False para desactivarlas.

 

Ejemplo:

 

# INFBASCLI.FMT
# Listado básico de clientes
#
<config>
StartConfString=$C10$$COFF$    // 10 CPI sin compresión (80 cols)
PaperLenght=60
TopMargin=2
LeftMargin=5                   // Margen para encuadernación
AutoEject=Yes
EjectAfterTitle=True
EjectBeforeSummary=False
</config>
