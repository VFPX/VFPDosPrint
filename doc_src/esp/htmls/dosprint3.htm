1. Introducción

 

La clase DOSPrint permite generar reportes en formato texto que aprovechan todas las características de las impresoras matriciales.

 

Esta nueva versión 3.0 incorpora un salto sustancial en este sentido al permitir el uso de archivos de formato, los cuales permiten definir "visualmente" el informe que se desea generar de una forma sencilla pero a la vez potente. Esto permite minimizar sensiblemente el código necesario para generar un informe basado en DOS, tal como se puede apreciar en los ejemplos incluidos.

 

 

 

 

2. Contenido del archivo DOSPRINT3.ZIP

El archivo DOSPRINT3.ZIP debe contener los siguientes archivos:

 

DOSPRINT3.VCX
DOSPRINT3.VCT		 Libreria de clases DOSPRINT
CLIENTES.DBF Tabla de clientes para el ejemplo INFBASCLI.PRG
INFBASCLI.FMT Formato de impresión utilizado en el ejemplo INFBASCLI.PRG
INFBASCLI.PRG Informe de ejemplo.
MOVCAJAS.DBF Tabla de movimientos de caja para el ejemplo MOVCAJAS.PRG
MOVCAJAS.FMT Formato de impresión utilizado en el ejemplo MOVCAJAS.PRG
MOVCAJAS.PRG Informe de ejemplo.
BLANK.FMT Formato en blanco que puede ser usado como base para nuevos informes.
Carpeta DOC Documentación de DOSPrint3 (ver DOSPRINT3.HTM)

 

 

 

 

3. Interfaz de la clase cDOSPrint

La interfaz de la clase cDOSPrint ha sido ampliada notablemente, a fín de permitir un mejor control sobre el proceso de impresión en los casos en los que se desea generar manualmente el informe.

 

Propiedades
AutoEject BottomMargin PrintJobName DetailString EjectAfterTitle
EjectBeforeSummary EjectString FooterString HeaderString LeftMargin
LineNo Macros[...] NewPageConfString OnNewPage PageNo
PaperLenght PrintFormat StartConfString SummaryString TopMargin
TitleString        

 

Métodos
AddMacro Clear Eject EndGroups EvalGroups
GetFormatSection GetMacro Print PrintSection PrintToFile
Run SectionHeight SetMacro StartGroups STRExpand
Write WriteLn

 

 

Las siguientes propiedades, aunque soportadas aún en esta versión, serán elminadas en la próxima versión de DOSPrint:

 

cDocName Sustituido por la propiedad PrintJobName
FooterLenght Sustituido por el método SectionHeight()
