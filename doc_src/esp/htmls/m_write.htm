Write (Método)

Imprime un texto en el informe actual sin saltar a una nueva linea.

Aplica a:


Clase DOSPrint

Sintaxis:


Object.Write(cText[,lLeftMargin])

Donde:

 

Object Referencia a un objeto perteneciente a una de las clases indicadas en Aplica a:
cText Texto a imprimir. Note que el método no generará saltos de linea a menos que cText contenga más de una linea. En ese caso, se generará un salto de linea para cada linea en cText, menos para la última.
lLeftMargin Opcional. Permite indicar si se desea imprimir el texto respetando el márgen izquierdo establecido en la propiedad LeftMargin. Si no se indica, el texto se imprimirá directamente al lado del último texto impreso.
