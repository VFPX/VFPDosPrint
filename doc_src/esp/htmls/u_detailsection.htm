Sección DETAIL

La sección DETAIL define el texto que se mostrará en cada linea de detalle del informe. Esta sección normalmente contiene expresiones expandibles que corresponden a las columnas del informe.

Ejemplo:

 

# INFBASCLI.FMT
# Listado básico de clientes
#
<config>
StartConfString=$C10$$COFF$    // 10 CPI sin compresión (80 cols)
PaperLenght=60
TopMargin=2
LeftMargin=5                   // Margen para encuadernación
</config>

<macros>
CSTATUS=proper(STATUS)
EMPRESA='Tuberias XYZ'
DIRECCIONEMP='Caracas, Venezuela'
</macros>

<header>
#...+....[....+....[....+....1....+....1....+....1....+....1....+....1....+....1
[EMPRESA                          ]                   LISTADO BASICO DE CLIENTES
[DIRECCIONEMP                     ]

CODIGO                NOMBRE                                 STATUS
====================  ====================================== ==========
</header>

<detail>
[CODIGO            ]  [PROPER(NOMBRE)                      ] [CSTATUS]
</detail>


<footer>


[datetime           ]                                                   [pageno]
</footer>
