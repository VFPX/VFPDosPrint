WriteLn (Método)

Imprime un texto en el informe actual y salta a una nueva linea. Este método normalmente es llamado internamente por otros métodos y no se recomienda su uso extensivo. En su lugar, utilize el método PrintSection o el método Run.

Aplica a:


Clase DOSPrint

Sintaxis:


Object.WriteLn(cText[,lDoCals][,lNoEject])

Donde:

 

Object Referencia a un objeto perteneciente a una de las clases indicadas en Aplica a:
cText Texto a imprimir. Si el texto contiene más de una linea, se generará un salto de linea por cada una de ellas.
lDoCalcs Opcional. Permite indicar si se desea ejecutar las funciones de totalización al imprimir la linea.
lNoEject Opcional. Si se coloca en Verdadero, no se ejecutarán saltos de página si el texto a imprimir sobrepasa el tamaño actual de la página.
