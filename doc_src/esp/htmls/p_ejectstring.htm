EjectString (Propiedad)

Permite establecer la cadena que se utilizará para producir un salto de página. El valor por defecto de esta propiedad es el caracter 12 (CHAR 12). Al configurar esta propiedad, se puede utilizar la constante especial $CRLF$ que representa un salto de linea (ver ejemplo).

Ejemplo(s):

*-- Se configura el reporte para que avance 4 lineas cuando salte de página.
*
#DEFOBJ oRep AS DOSPrint
oRep.PaperLenght:=20
oRep.EjectString:=REPLICATE($CRLF$,4)
...
