SummaryString (Propiedad)

Permite indicar el texto a imprimir como resumen final del informe. La banda de resumen (SUMMARY) se emite una sola vez al finalizar el informe (vea propiedad EjectBeforeSummary). Si se está utilizando un formato de impresión, esta variable contiene el texto indicado en la sección SUMMARY.
