TitleString (Propiedad)

Permite indicar el texto a imprimir como título general del informe. La banda de título (TITLE) se emite una sola vez antes de empezar la primera página (vea propiedad EjectAfterTitle). Si se está utilizando un formato de impresión, esta variable contiene el texto indicado en la sección TITLE.
