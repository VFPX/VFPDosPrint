Sección FOOTER

La sección FOOTER define el texto que se mostrará como pié de cada página. Esta sección normalmente contiene información como la hora y fecha del informe, así como el nro. de página actual (a través del macro $PAGENO$). Si la propiedad PaperLenght es igual a cero (0), esta sección se imprimirá inmediatamente después de la última linea de detalle del informe. Si PaperLenght contiene cualquier valor diferente de cero, la sección se imprimirá al final de cada página en una posición fija.

Ejemplo:

 

# INFBASCLI.FMT
# Listado básico de clientes
#
<config>
StartConfString=$C10$$COFF$    // 10 CPI sin compresión (80 cols)
PaperLenght=60
TopMargin=2
LeftMargin=5                   // Margen para encuadernación
</config>

<macros>
CSTATUS=proper(STATUS)
EMPRESA='Tuberias XYZ'
DIRECCIONEMP='Caracas, Venezuela'
</macros>

<header>
#...+....[....+....[....+....1....+....1....+....1....+....1....+....1....+....1
[EMPRESA                          ]                   LISTADO BASICO DE CLIENTES
[DIRECCIONEMP                     ]

CODIGO                NOMBRE                                 STATUS
====================  ====================================== ==========
</header>

<detail>
[CODIGO            ]  [PROPER(NOMBRE)                      ] [CSTATUS]
</detail>


<footer>


[datetime           ]                                                   [pageno]
</footer>
