SectionBeginMark(Propiedad)

Permite establecer la cadena de caracteres que se tomará como inicio de sección en un archivo de formato. Esto es útil cuando se utiliza DOSPrint para generar documentos con formato, tales como HTML, XML, etc.

El valor por defecto de esta propiedad es "<".

Ejemplo(s):


<config>
SectionBeginMark='['
SectionEndMark=']'
</config>

[header]
<html>
<head><title>Prueba</title>
</head>
<body>
[/header]

[summary]
</body>
</html>
[/summary]
