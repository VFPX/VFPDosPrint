Sección TITLE

La sección TITLE identifica el texto a imprimir como presentación del informe. La sección TITLE se imprime solo una vez antes de imprimir la primera página del informe.   Si la propiedad EjectAfterTitle está en Verdadero, se emitirá un salto de página después de imprimir esta sección.

Ejemplo:

 

# INFBASCLI.FMT
# Listado básico de clientes
#
<config>
StartConfString=$C10$$COFF$    // 10 CPI sin compresión (80 cols)
PaperLenght=60
TopMargin=2
LeftMargin=5                   // Margen para encuadernación
EjectAfterTitle=Yes
</config>

<macros>
CSTATUS=proper(STATUS)
EMPRESA='Tuberias XYZ'
DIRECCIONEMP='Caracas, Venezuela'
NUMCLI=TRANS(_COUNT[CODIGO],"99,999")
</macros>

<title>


                                  TUBERIAS XYZ

 

                      $BON$INFORME BASICO DE CLIENTES$BOFF$

                                    AL $DATE$

 

 

</title>

 

 

[Regresar]
