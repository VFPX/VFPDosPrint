GetMacro (Método)

Retorna la expresión (sin evaluar) o el valor asociado a una macro definida previamente.

Aplica a:


Clase DOSPrint

Sintaxis:


Object.GetMacro(cName)

Donde:

Object Referencia a un objeto perteneciente a algunas de las clases indicadas en Aplica a:
cName Nombre del macro (sin los signos $) cuya expresión se desea obtener. Note que si la macro indicada no existe, el método devuelve la cadena vacia.

Ejemplo(s):

*-- Se obtiene la definición del macro C10
*
cC10:=oRep.GetMacro("C10")
