StartConfString (Propiedad)

Permite establecer la cadena que se enviará a la impresora al iniciar el informe. Esta cadena puede contener los códigos necesarios para configurar la impresora según las necesidades del informe. Dentro de esta propiedad puede utilizar las siguientes macros predefinidas:

 

$C10$ Configurar la impresora a 10 CPI
$C12$ Configurar la impresora a 12 CPI
$CON$ Activar la compresión
$COFF$ Desactivar la compresión

 

Aplica a:


Clase DOSPrint

Ejemplo(s):

*-- Se configura el informe para que imprima a 12 CPI
*
#DEFOBJ oRep AS DOSPrint
oRep.PaperLenght:=60
oRep.StartConfString:=$C12$
...
