Clear (Método)

Realiza las acciones necesarias para iniciar un nuevo informe. Específicamente, este método realiza las siguientes acciones:

 

  • Inicializa las propiedades PageNo y LineNo
  • Limpia el buffer local de impresión
  • Inicializa las variables internas utilizadas por las funciones de acumulación.

 

Aplica a:


Clase DOSPrint

Sintaxis:


Object.Clear()
