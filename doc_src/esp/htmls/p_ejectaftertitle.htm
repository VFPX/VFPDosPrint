EjectAfterTitle (Propiedad)

Permite indicar si se emitirá un salto de página luego de imprimir la sección TITLE. El valor por defecto de esta propiedad es Verdadero. Note que el valor de la propiedad AutoEject tiene prioridad sobre esta propiedad, por lo que si AutoEject contiene el valor Falso, no se saltará la página aun cuando EjectAfterTitle sea Verdadero.
