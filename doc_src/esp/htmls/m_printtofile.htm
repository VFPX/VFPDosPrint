PrintToFile (Método)

Envia el informe actual hacia un archivo cuyo nombre se indica.

Aplica a:


Clase DOSPrint

Sintaxis:


Object.PrintToFile(cFileName[,nCopies])

Donde:

 

Object Referencia a un objeto perteneciente a una de las clases indicadas en Aplica a:
cFileName Nombre del archivo de salida. Note que si el nombre del archivo indicado es inválido o el directorio indicado no existe, se producirá un error en tiempo de ejecución.
nCopies Opcional. Nro. de copias a generar del informe. Si no se indica, se asume 1.

Ejemplo(s):

*-- Se genera el informe
*
oRep.StartGroups()
#DOWHILE NOT EOF()
 oRep.EvalGroups()
 oRep.PrintSection("DETAIL")
 #DBSKIP
#ENDDO
oRep.EndGroups()


*-- Se graba el informe un archivo para su posterior envio
*
#DEFVAR cFile
cFile:=Kernel.DLGS.GetFile("REP","Indique el archivo de salida:","Grabar informe")
#LET oRep.PrintToFile(cFile) ON not empty(cFile)
