Print (Método)

Envia el informe actual hacia la impresora predeterminada o hacia una impresora específica. El método retorna Verdadero si el informe pudo ser colocado en la impresora o Falso si ocurrió algún error con la cola de Windows.

 

Aplica a:


Clase DOSPrint

Sintaxis:


Object.Print([cPrinter][,nCopies])

Donde:

 

Object Referencia a un objeto perteneciente a una de las clases indicadas en Aplica a:
cPrinter Opcional. Nombre de la impresora a donde se desea enviar el informe. Si no se indica, se asume la impresora predeterminada.
nCopies Opcional. Nro. de copias a imprimir del informe. Si no se indica, se asume 1.

Ejemplo(s):

*-- Se imprime el informe (original y copia)
*
oRep.StartGroups()
#DOWHILE NOT EOF()
 oRep.EvalGroups()
 oRep.PrintSection("DETAIL")
 #DBSKIP
#ENDDO
oRep.EndGroups()
oRep.Print("Epson FX-1050",2)
