PrintSection (Método)

Imprime una sección indicada. Normalmente, este método no es utilizado si se está usando un archivo de formato.

Aplica a:


Clase DOSPrint

Sintaxis:


Object.PrintSection(cSeccion[,lNoEject])

Donde:

 

Object Referencia a un objeto perteneciente a una de las clases indicadas en Aplica a:
cSeccion Nombre de la sección a imprimir. Si se indica la sección TITLE, HEADER, DETAIL, FOOTER o SUMMARY, se utilizará el valor de las propiedades TitleString, HeaderString, DetailString, FooterString y SummaryString respectivamente; de lo contrario, se buscará la sección indicada en el archivo de formato actual.
lNoEject Opcional. Permite indicar que no se desea efectuar saltos de página mientras se imprime la sección. Especialmente útil cuando se desea imprimir manualmente la sección FOOTER.

Ejemplo(s):

*-- Se genera el informe
*
oRep.StartGroups()
#DOWHILE NOT EOF()
 oRep.EvalGroups()
 oRep.PrintSection("DETAIL")
 #DBSKIP
#ENDDO
oRep.EndGroups()
