EjectBeforeSummary (Propiedad)

Permite indicar si se emitirá un salto de página antes de imprimir la sección SUMMARY. El valor por defecto de esta propiedad es Verdadero. Note que el valor de la propiedad AutoEject tiene prioridad sobre esta propiedad, por lo que si AutoEject contiene el valor Falso, no se saltará la página aun cuando EjectBeforeSummary sea Verdadero.
