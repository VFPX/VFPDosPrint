HeaderString (Propiedad)

Permite establecer el texto a imprimir en la banda de encabezado del informe. Si se está usando un formato de impresión, esta propiedad contendrá el texto indicado en la sección HEADER del formato indicado. Note que aunque la propiedad es de lectura/escritura, el hecho de cambiar el valor de la propiedad NO afecta al formato indicado.

Aplica a:


Clase DOSPrint

Ejemplo(s):

*-- Se configura el reporte
*
#DEFOBJ oRep AS DOSPrint
oRep.PaperLenght:=60
oRep.HeaderString:="INFORME DE CLIENTES $CRLF$" + ;
                   "AL $DATE$"
oRep.DetailString:="[CODIGO] [NOMBRE              ]"
oRep.FooterString:="$DATETIME$                        $PAGENO$"
...

*-- Se imprimen los registros
*
#DBSELECT qClientes
#DBGO TOP
#DOWHILE Not EOF()
 oRep.PrintSection("DETAIL")
 #DBSKIP
#ENDDO
..
