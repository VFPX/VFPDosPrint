Sección HEADER

La sección HEADER define el texto que se mostrará como encabezado de cada página. Esta sección normalmente contiene las lineas de identificación del informe así como los títulos de las columnas.

Ejemplo:

 

# INFBASCLI.FMT
# Listado básico de clientes
#
<config>
StartConfString=$C10$$COFF$    // 10 CPI sin compresión (80 cols)
PaperLenght=60
TopMargin=2
LeftMargin=5                   // Margen para encuadernación
</config>

<macros>
CSTATUS=proper(STATUS)
EMPRESA='Tuberias XYZ'
DIRECCIONEMP='Caracas, Venezuela'
</macros>

<header>
#...+....[....+....[....+....1....+....1....+....1....+....1....+....1....+....1
[EMPRESA                          ]                   LISTADO BASICO DE CLIENTES
[DIRECCIONEMP                     ]

CODIGO                NOMBRE                                 STATUS
====================  ====================================== ==========
</header>
