NewPageConfString (Propiedad)

Permite establecer la cadena que se enviará a la impresora cada vez que se inicie una nueva página. Esta cadena puede contener los códigos necesarios para configurar la impresora según las necesidades del informe. Dentro de esta propiedad puede utilizar las mismas macros indicadas para la propiedad StartConfString.

 

Ejemplo(s):

*-- Se configura el informe para que imprima a 12 CPI
*
#DEFOBJ oRep AS DOSPrint
oRep.PaperLenght:=60
oRep.NewConfString:=$C12$
...
