Sección MACROS

La sección MACROS define los diferentes macros que estarán disponibles en el informe. Cada linea en la sección corresponde a un par del tipo Macro=Expresión, donde Macro corresponde al nombre del macro y Expresión a la expresión o constante que define el macro. Una definición de macro puede tener diferentes sintáxis:

Tipo Ejemplo
Constante EMPRESA='Tuberias XYZ'
PI=3.1416
Expresión simple CSTATUS=PROPER(QClientes.STATUS)
Expresión expandible CSTATUS='Status actual: %QClientes.Status%'

 

Adicionalmente, una definición de macro puede ser usada para realizar operaciones de acumulación, estadística o conteo. La clase DOSPrint reconoce las siguientes funciones de acumulación:

Función Parámetros Ejemplo
_SUM
 nExpr: Valor a sumar
cReset: Reiniciar el acumulador cuando este valor cambie.
nInitVal: Valor inicial		 TMONTO=_SUM[MONTO;CODCAJA;0.0]
_MIN
 nExpr: Valor a comparar
cReset: Reiniciar el acumulador cuando este valor cambie.
nInitVal: Valor inicial		 TMINPAGO=_MIN[MONTO;CODCAJA]
_MAX
 nExpr: Valor a comparar
cReset: Reiniciar el acumulador cuando este valor cambie.
nInitVal: Valor inicial		 TMAXPAGO=_MAX[MONTO;CODCAJA]
_AVG
 nExpr: Valor a comparar
cReset: Reiniciar el acumulador cuando este valor cambie.
nInitVal: Valor inicial		 TPAGOPROM=_AVG[MONTO;CODCAJA]
_COUNT cReset: Reiniciar el contador cuando este valor cambie.
nInitVal: Valor inicial		 TNUMPAGOS=_COUNT[CODCAJA]

 

En cada caso, si no se indica el parámetro cReset, la operación se realizará hasta el final del informe. Si se desea realizar un cálculo que se reinicie en cada página, utilize la constante PAGENO como valor para cReset.

Ejemplo:

 

# INFBASCLI.FMT
# Listado básico de clientes
#
<config>
StartConfString=$C10$$COFF$    // 10 CPI sin compresión (80 cols)
PaperLenght=60
TopMargin=2
LeftMargin=5                   // Margen para encuadernación
EjectBeforeSummary=No
</config>

<macros>
CSTATUS=proper(STATUS)
EMPRESA='Tuberias XYZ'
DIRECCIONEMP='Caracas, Venezuela'
NUMEMP=TRANS(_COUNT[],"9,999")
</macros>

<header>

[DIRECCIONEMP                     ]

CODIGO                NOMBRE                                 STATUS
====================  ====================================== ==========
</header>

<detail>
[CODIGO            ]  [PROPER(NOMBRE)                      ] [CSTATUS]
</detail>


<footer>


[datetime           ]                                                   [pageno]
</footer>

<summary>

************************
TOTAL CLIENTES: [NUMCLI]
************************
</summary>
