EvalGroups (Método)

Evalua los grupos definidos y genera los saltos de grupo necesarios. Este método solo aplica si se está usando un archivo de formato.

 

Aplica a:


Clase DOSPrint

Sintaxis:


Object.EvalGroups([lEOF])

Donde:

 

Object Referencia a un objeto perteneciente a una de las clases indicadas en Aplica a:
lEOF Opcional. Valor lógico que indica si se está en el final del informe. Si se indica, se imprimirán los pie de grupos pero no las cabezera de grupo. Esto es equivalente al método EndGroups.

Ejemplo(s):

*-- Se imprime el informe
*
oRep.StartGroups()
#DOWHILE NOT EOF()
 oRep.EvalGroups()
 oRep.PrintSection("DEATAIL")
 #DBSKIP
#ENDDO
oRep.EndGroups()
