EndGroups (Método)

Evalua los grupos definidos y genera los saltos de grupo necesarios para finalizar el informe. Este método solo aplica si se está usando un archivo de formato. Este método es equivalente al método EvalGroups con el parámetro lEOF en Verdadero.

 

Aplica a:


Clase DOSPrint

Sintaxis:


Object.EndGroups()

Donde:

 

Object Referencia a un objeto perteneciente a una de las clases indicadas en Aplica a:

Ejemplo(s):

*-- Se imprime el informe
*
oRep.StartGroups()
#DOWHILE NOT EOF()
 oRep.EvalGroups()
 oRep.PrintSection("DEATAIL")
 #DBSKIP
#ENDDO
oRep.EndGroups()
