PaperLenght (Propiedad)

Permite indicar el tamaño del papel a utilizar (en lineas). Note que esta propiedad es utilizada para determinar los puntos donde se debe saltar a una nueva página, pero no para configurar físicamente el tamaño de la página en la impresora (para más información sobre como configurar el tamaño de página en la impresora, consulte el manual de usuario de su impresora). Para configurar el tamaño de la página en su impresora, puede enviar el comando necesario usando la propiedad StartConfString.

 

Si el informe se emitirá en un papel continuo sin saltos de página, como en ciertas impresoras fiscales, puede colocarse PaperLenght a cero, lo cual indica a la clase que el papel no tiene un tamaño determinado sino que es continuo. Esto causará que el informe imprima todas las lineas de detalle sin saltar la página y que el pié de página se imprima justo al finalizar la última linea de detalle.

Aplica a:


Clase DOSPrint

Ejemplo(s):

*-- Se configura la página del informe
*
#DEFOBJ oRep AS DOSPrint
oRep.PaperLenght:=60
oRep.TopMargin:=2
oRep.BottomMargin:=2
oRep.LeftMargin:=5
