Run (Método)

Imprime un informe completo utilizando el archivo de formato indicado en la propiedad PrintFormat.

Aplica a:


Clase DOSPrint

Sintaxis:


Object.Run([cAlias][,cWhileExpr][,cForExpr][,nDSID])

Donde:

 

Object Referencia a un objeto perteneciente a una de las clases indicadas en Aplica a:
cAlias Opcional. Alias de la tabla, cursor o vista que se recorrerá para el informe. Si no se indica, se asume el alias activo.
cWhileExpr Opcional. Expresión condicional que indica la condición que debe cumplirse para que se continue con la generación del informe. El informe se detendrá en el primer registro que no cumpla con la condición indicada en este parámetro. Si no se indica, se procesan todos los registros del alias a recorrer.
cForExpr Opcional. Expresión condicional que indica la condición que debe cumplir un registro para ser incluido dentro del informe. Note que el cuando se encuentre un registro que no cumpla con esta condición, el informe no se detendrá sino que continuará con el siguiente registro en el alias.
nDSID Opcional. Sesión de datos a utilizar para el informe. Si no se indica, se asume la sesión de datos predeterminada.

Ejemplo(s):

*-- Se genera el informe
*
#DEFOBJ oRep AS DOSPrint
oRep.PrintFormat:="INFBASCLI.FMT"
oRep.Clear()
oRep.Run("QCLIENTES","","STATUS=[ACTIVO]")
oRep.Print()
#UNDEFOBJ oRep
