GetFormatSection (Método)

Devuelve el contenido de una sección cualquiera en el archivo de formato actual. Este método solo aplica si se está usando un archivo de formato.

 

Aplica a:


Clase DOSPrint

Sintaxis:


Object.GetFormatSection(cSeccion)

Donde:

 

Object Referencia a un objeto perteneciente a una de las clases indicadas en Aplica a:
cSeccion Nombre de la sección del archivo de formato que se desea obtener.
