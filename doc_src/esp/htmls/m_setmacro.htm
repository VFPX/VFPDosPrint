SetMacro (Método)

Permite cambiar la expresión asociada a un macro definido previamente.

Aplica a:


Clase DOSPrint

Sintaxis:


Object.SetMacro(cName,cExpr)

Donde:

Object Referencia a un objeto perteneciente a algunas de las clases indicadas en Aplica a:
cName Nombre del macro (sin los signos $) cuya expresión se desea cambiar. Note que si el macro indicado no existe, no se generará un error.
cExpr Nueva expresión o valor del macro.

Ejemplo(s):

*-- Se actualiza el macro PAGENO
*
oRep.SetMacro("PAGENO",nPageNo)
