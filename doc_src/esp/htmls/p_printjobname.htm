PrintJobName (Propiedad)

Permite establecer el nombre que se dará al informe cuando sea colocado en la cola de impresión de Windows. Si no se indica, se asumirá el nombre Documento VFP DosPrint.

Ejemplo(s):

*-- Se imprime el informe
*
oRep.PrintJobName:="Factura #"+cNumFac
oRep.Print()
