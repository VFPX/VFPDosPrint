Sección SUMMARY

La sección SUMMARY identifica el texto a imprimir como resumen final del informe. La sección SUMMARY se imprime solo una vez al finalizar el informe.  Si la propiedad EjectBeforeSummary está en Verdadero, se emitirá un salto de página antes de imprimir esta sección.

Ejemplo:

 

# INFBASCLI.FMT
# Listado básico de clientes
#
<config>
StartConfString=$C10$$COFF$    // 10 CPI sin compresión (80 cols)
PaperLenght=60
TopMargin=2
LeftMargin=5                   // Margen para encuadernación
</config>

<macros>
CSTATUS=proper(STATUS)
EMPRESA='Tuberias XYZ'
DIRECCIONEMP='Caracas, Venezuela'
NUMCLI=TRANS(_COUNT[CODIGO],"99,999")
</macros>

<summary>

 

***********************************
TOTAL CLIENTES EN ARCHIVO: [NUMCLI]

 

 

</summary>

 

 

[Regresar]
