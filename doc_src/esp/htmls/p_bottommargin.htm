BottomMargin (Propiedad)

Permite indicar el margen inferior (en lineas) del informe.

Ejemplo(s):

*-- Se configura la página del informe
*
#DEFOBJ oRep AS DOSPrint
oRep.PaperLenght:=60
oRep.TopMargin:=2
oRep.BottomMargin:=2
oRep.LeftMargin:=5
