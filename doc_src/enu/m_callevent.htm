CallEvent (Method)

Executes an event defined in the current format file. The result of the event is inserted in the current report.

Applies to:


DOSPrint class

Syntax:


Object.CallEvent(cEvent[,pu0...pu10])

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cEvent Name of the event to execute. If the event does not exists, an empty string will be returned.
pu0...pu10 Up to 10 optional arguments to be passed to the event.

Examples:

*-- How to use internal events to do calculations inside a format
*
<detail>
[COD] [CUSTNAME         ] [TINVOICES    ]
#SET TINVOICES=DP.CallEvent('GetCustomerInvoices',COD)
</detail>

# GetCustomerInvoices (Event)
# Return the total amout for customer's invoices. DPArg0 and DPRETURN are predefined 
# variables.
#
<GetCustomerInvoices Type=EVENT>
LOCAL cCustID,nTotal
cCustID=DPArg0

SELECT 0
SELECT SUM(Total) AS Total FROM Invoices WHERE CustID=cCustID INTO CURSOR Q1
nTotal=Q1.Total
USE IN Q1

DPRETURN=TRANSFORM(nTotal,"9,999,999.99")+"$"
</GetCustomerInvoices>
