Sección FOOTER

Defines the text to be printed at the bottom of any page, or at the end of the report if PaperLenght is cero.

Example:

# CUSTOMERS.FMT
# Basic customer report
#
<config>
StartConfString=$C10$$COFF$    // 10 CPI not condensed (80 cols)
PaperLenght=60
TopMargin=2
LeftMargin=5                 
</config>

<macros>
CSTATUS=proper(STATUS)
COMPNAME='XYZ Bookstore'
COMPADDRESS='Caracas, Venezuela'
</macros>

<header>
#...+....[....+....[....+....1....+....1....+....1....+....1....+....1....+....1
[COMPNAME                         ]                                CUSTOMER LIST
[COMPADDRESS                      ]

CUSTOMER ID           CUSTOMER NAME                          STATUS
====================  ====================================== ==========
</header>

<detail>
[COD               ]  [PROPER(CUSTNAME)                    ] [CSTATUS]
</detail>


<footer>


[datetime           ]                                                   [pageno]
</footer>
