GetFormatSectionList (Method)

Returns a collection with the names of all sections contained in the current format file.

Applies to:


DOSPrint class

Syntax:


Object.GetFormatSectionList

Where:

Object Reference to an object of one of the class mentioned in Applies to:

Examples:

*-- List all sections in a format file
*
oDP.PrintFormat="CUSTOMERS.FMT"
oSectionList=oDP.GetFormatSectionList()
FOR i=1 TO oSectionList.Count
 ?oSectionList.Items(i)
ENDFOR
