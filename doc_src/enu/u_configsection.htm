CONFIG Section

This section is used to setup any report's options like page size, margins, printer configuration, etc. Each line in this sections must be a Property=Value pair, where Property is the name of the class's property to setup and Value is the new value of the property. For boolean properties (as AutoEject, EjectAfterTitle, etc), you can use any of the following constants to set its value:

True False
True | Yes | .T. False | No | .F.

Note that if a non existing property is indicated in this sections, it will be ignored without generating any error.

Example:

# CUSTOMERS.FMT
# Basic customer listing
#
<config>
StartConfString=$C10$$COFF$    // Non-condensed 10 CPI (80 cols)
PaperLenght=60                 // Letter size
TopMargin=2
LeftMargin=5
AutoEject=Yes
EjectAfterTitle=True
EjectBeforeSummary=False
</config>
