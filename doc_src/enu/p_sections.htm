Sections (Collection)

A collection of Section objects. Each item in the collection describes a section in the current format file.

Applies to:


DOSPrint class

Example:

*-- Prints text of all sections in a format file
*
oDP=CREATEOBJECT("cDOSPrint")
oDP.FormatFile="CUSTOMERS.FMT"
FOR i=1 TO oDP.Sections.Count
 ?oDP.Sections.Items(i).Text,
ENDFOR
