PrintSection (Method)

Prints an specified section. Normally, this method is not used if you are using format files.

Applies to:


DOSPrint class

Syntax:


Object.PrintSection(cSeccion[,lNoEject])

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cSeccion Name of the section to print. If the secions indicated is TITLE, HEADER, DETAIL, FOOTER or SUMMARY, the corresponding properties (TitleString, HeaderString, DetailString and FooterString) will be used; in any other case, the section will be searched in the current format file.
lNoEject Optional. If True, no page breaks will be executed while printing the section.

Examples:

oRep.StartGroups()
DO WHILE NOT EOF()
 oRep.EvalGroups()
 oRep.PrintSection("DETAIL")
 SKIP
ENDDO
oRep.EndGroups()

See also:


Write, WriteLn, Format files
