BandHeight (Property)

Determines the height (in lines) of the section. This property applies if:

a) Its value is greater than zero (default value)
b) Its value is greater than the actual height of section.
c) IntegralHeight property is setted to True.

This property can be used to ensure that a group header section is printed in the current page only if there is enough space to print at least one detail line of that group.

Applies to:


Section object

Example:

<groups>
g1 = rectype | g1hdr | g1ftr
</groups>
...
<g1ftr type=g1hdr IntegralHeight=True  BandHeight=5>

------------------
%rectype%
------------------
</g1ftr>
