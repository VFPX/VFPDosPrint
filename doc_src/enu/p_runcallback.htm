RunCallBack (Property)
(only available in registered version and when used inside VFP)

This property lets you to define a VFP function or method to be called at some points inside Run method. The function or method indicated here will be called with the following parameters:

pnMode: 1=Starting report, 2=Processing report, 3=Report finished
pnPageCount: Pages generated so far.

Applies to:


DOSPrint class

Example:

*-- Example of a RunCallBack
*
oDP=CREATEOBJECT("cDosPrint")
oDP.RunCallBack='MyProgress'
oDP.PrintFormat='CUSTOMERS.FMT'
oDp.Run()
...
*-- MyProgress
*   Custom progress message
*
PROC MyProgress(pnMode,pnPageCount)
 DO CASE
    CASE pnMode=1
         WAIT WINDOW "Preparing report..." NOWAIT
    CASE pnMode=2
         WAIT WINDOW "Printing...(" + ALLT(STR(pnPageCount))+" pages)" WINDOW NOWAIT
    CASE pnMode=3
         WAIT CLEAR
 ENDCASE
ENDPROC
