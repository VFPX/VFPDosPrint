NewPageConfString (Property)

Sets a string to be printed on every new page. This property is usefull to send control characters to config the printer.

Applies to:


DOSPrint class

Examples:

*-- Configure report for 12 CPI printing
*
oRep=CREATEOBJECT("cDOSPrint")
oRep.PaperLenght=60
oRep.NewConfString=$C12$
...

See also:


EjectString, StartConfString
