SendTextToPrinter (Method)

Sends a text to the specified printer.

Applies to:


DOSPrint class

Syntax:


Object.SendTextToPrinter(cText,cPrinter,cJobName)

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cText Text to be printed.
cPrinter Target printer.
cJobName Jobname to be shown in Windows printer's queue.
