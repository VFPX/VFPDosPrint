PrintToFile (Method)

Sends the current report to a disk file.

Applies to:


DOSPrint class

Syntax:


Object.PrintToFile(cFileName[,nCopies])

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cFileName Name of the disk file to generate. If this file is invalid or the folder does not exists, a runtime error will be generated.
nCopies Optional. Number of copies to generate. Copies property will be used if this parameter is not indicated.

Examples:

*-- Generate the report
*
oRep.PrintFormat="CUSTOMERS.FMT"
oRep.Run()


*-- Save the report to a file
*
LOCAL cFile
cFile=GetFile()
IF NOT EMPTY(cFile)
 oRep.PrintToFile(cFile)
ENDIF

See also:


Print, SendToFile
