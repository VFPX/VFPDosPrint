TITLE Section

Defines the text to be printed at the beginning of the report. The contents of this section is printed once at the beginning of the report. If EjectAfterTitle is setted to True, a page break will be generated after printing this section.

Example:

# CUSTOMERS.FMT
# Basic customer report
#
<config>
StartConfString=$C10$$COFF$    // 10 CPI not condensed (80 cols)
PaperLenght=60
TopMargin=2
LeftMargin=5
EjectAfterTitle=True
</config>

<macros>
CSTATUS=proper(STATUS)
COMPNAME='XYZ Bookstore'
COMPADDRESS='Caracas, Venezuela'
</macros>

<title>
#...+....[....+....[....+....1....+....1....+....1....+....1....+....1....+....1
%PADC(COMPNAME,80)%
%PADC(COMPADDRESS,80)%





                            BASIC CUSTOMER LISTING

                            Date: {DATE}
			    Time: {TIME}
                            User: {SYS(0)}


</title>


<header>
#...+....[....+....[....+....1....+....1....+....1....+....1....+....1....+....1
[COMPNAME                         ]                                CUSTOMER LIST
[COMPADDRESS                      ]

CUSTOMER ID           CUSTOMER NAME                          STATUS
====================  ====================================== ==========
</header>

<detail>
[COD               ]  [PROPER(CUSTNAME)                    ] [CSTATUS]
</detail>


<footer>


[datetime           ]                                                   [pageno]
</footer>
