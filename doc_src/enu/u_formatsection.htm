FORMAT Section

This section is designed as a simplified way to define the basic layout of a report. Each line inside this sections starts with a two-chars code, followed by a ':'  and a space char,  wich defines the layout section the line belongs to. Note that if you use this section inside you format file, you can not use the normal layout sections, as Header, Detail, etc.

This is the list of two-chars codes and the layout sections associated:

Code Associated section
ti: TITLE
he: HEADER
de: DETAIL
fo: FOOTER
su: SUMMARY

Example:

# CUSTOMERS.FMT
# Basic customer report (FORMAT version)
#
<config>
StartConfString=$C10$$COFF$    // 10 CPI not condensed (80 cols)
PaperLenght=60
TopMargin=2
LeftMargin=5                 
</config>

<macros>
CSTATUS=proper(STATUS)
COMPNAME='XYZ Bookstore'
COMPADDRESS='Caracas, Venezuela'
</macros>

<format>
#...+....[....+....[....+....1....+....1....+....1....+....1....+....1....+....1
he: [COMPNAME                         ]                                CUSTOMER LIST
he: [COMPADDRESS                      ]
he: 
he: CUSTOMER ID           CUSTOMER NAME                          STATUS
he: ====================  ====================================== ==========
de: [COD               ]  [PROPER(CUSTNAME)                    ] [CSTATUS]
fo: 
fo: 
fo: 
fo: [datetime           ]                                                   [pageno]
</format>
