Clear (Method)

Restores the object to its initial state, in order to start a new report. This method does the following tasks:

  • Initialize PageNo and LineNo properties
  • Clear local print buffer
  • Initialize internal variables used within aggregate funcions like _SUM.

Applies to:


DOSPrint class

Syntax:


Object.Clear()

Where:

Object Reference to an object of one of the class mentioned in Applies to:
