RunFormat (Method)

Loads a format file and runs it.

Applies to:


DOSPrint class

Syntax:


Object.RunFormat(cFormat[,cAlias][,nPrintTo][,cPrinter][,cWhile][,cFor][,nDSID])

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cFormat Name of the format file to run.
cAlias Optional. Alias of the dataset to use or a reference to an ADO RecordSet object. If omitted, the current workarea will be used.
nPrintTo Optional. Destination of the report:

1 or ommited: Printer. cPrinter parameter or PrinterName will be used as the target printer.
2: Disk file. The destination file will be the one indicated in FileName property
3: Text
cPrinter Optional. Name of the printer to be used. If omitted, PrinterName property will be used.
cWhile Optional. If one of the record in the dataset fails to this condition, no further records will be evaluated and the report will be normally terminated. At this moment, this parameter is not used with ADO RecordSets.
cFor Optional. Any record in the dataset wich fails to this condition will be skipped and not included in the report. At this moment, this parameter is not used with ADO RecordSets.
nDSID Optional. Datasession ID to use. If omitted, the current datasession will be used. This parameter has no effect on ADO Recordsets.

Examples:

*-- Run the report and sent it to a printer
*
oDP=CREATEOBJECT("cDOSPrint")
oDP.RunFormat('CUSTOMERS.FMT','QCUST',1,'FX-1050')
