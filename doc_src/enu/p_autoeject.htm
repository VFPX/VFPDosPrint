AutoEject (Property)

Automatic page break generation. If True (default value), any page break required will be automatically generated by the class. If False, no page breaks will be generated, until you manually invoque Eject method.

Applies to:


DOSPrint class

Values:

True (Default) Eject method will be automatically invoqued when needed.
False Eject method will not be automatically invoqued. Note that when Autoeject is setted to False, line counter will be increased even when if it exceeds the current page size defined in PaperLenght property.

See also:


Eject, PaperLenght
