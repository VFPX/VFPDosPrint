EjectString (Property)

String to be sended to the printer to generate a page break. The default value for this property is CHR(12).

Applies to:


DOSPrint class

Examples:

*-- Configure class to skip 4 lines at page break
*
oRep=CREATEOBJECT("cDOSPrint")
oRep.PaperLenght=20
oRep.EjectString=REPLICATE($CRLF$,4)
...

See also:


StartConfString, NewPageConfString
