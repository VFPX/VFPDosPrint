Macros (Collection)

A collection of Macro objects. Each item in the collection describes a macro defined in a given section of the current format file.

Applies to:


Section object

Example:

*-- Shows all macros defined in DETAIL section
*
oDP=CREATEOBJECT("cDOSPrint")
oDP.FormatFile="CUSTOMERS.FMT"
oDetails=oDP.Sections.Items("DETAIL")
FOR i=1 TO oDetails.Macros.Count
 ?oDetails.Macros.Items(i).Name
ENDFOR
