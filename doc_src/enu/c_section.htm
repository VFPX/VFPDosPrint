Section (Object)

Each section in a format file is represented by a Section object. This objects are automatically created and added to the Sections collection when a format file is loaded by setting PrintFormat property. Note that you can change any property of a Section object, but those changes will not be saved to the original format file.

Applies to:


Sections collection

Properties
ApplyIf BandHeight IntegralHeight Macros Name
PrintIfBlank PrintOnNewPage ResetPageCounter StartOnNewPage Text
Type        

Example:

*-- Show some info for DETAIL section
*
oDP.PrintFormat="CUSTOMERS.FMT"
oSection=oDP.Sections.Items("DETAIL")
?oSection.Text               --> "[COD] [CUSTNAME              ] [CUSTADDRESS           ]"
?oSection.Type               --> "DETAIL"
?oSection.ApplyIf            --> .T.
?oSection.PrintOnNewPage     --> .F.
