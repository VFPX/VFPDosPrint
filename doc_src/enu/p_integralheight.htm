IntegralHeight (Property)

Determines wether or not a section will be printed in a new page if there is not enough space in the current page for the hole contents of the section. The default value for this property is False. This property is specially usefull for group footer's sections.

Applies to:


Section object

Example:

<groups>
g1 = rectype | g1hdr | g1ftr
</groups>
...
<g1ftr type=g1ftr IntegralHeight=True>
                       -----------------
              BALANCE: [TBALANCE       ]
              MINIMAL: [TMINVALUE      ]
              MAXIMAL: [TMAXVALUE      ]
              AVERAGE: [TAVGVALUE      ]
</g1ftr>
