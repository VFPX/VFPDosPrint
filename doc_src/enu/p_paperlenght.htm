PaperLenght (Property)

Page height in lines. This property is used to determinate how many lines fits on a single page and when to start a new page, but to phisically configures the printer's paper lenght (for more information about how to change your printer's page height, see your printer's User's Manual). You can use StartConfigString property to send any control character required phisically setup the paper height.

 

If your report will use continuos paper feed, you can set PaperLenght to cero. This way, the class will not generate any page break and will prints any page footer at the end of the report.

Applies to:


DOSPrint class

Examples:

*-- Report setup
*
oRep=CREATEOBJECT("cDOSPrint")
oRep.PaperLenght=60
oRep.TopMargin=2
oRep.BottomMargin=2
oRep.LeftMargin=5
