EvalGroups (Method)

Evaluates any defined group and generates any required group break. This method is valid only if you are using format files.

Applies to:


DOSPrint class

Syntax:


Object.EvalGroups([lEOF])

Where:

Object Reference to an object of one of the class mentioned in Applies to:
lEOF Optional. Boolean parameter used to indicate if you are at the end of the report. If True, group footers will be generated, without generaing group headers. This is equivalent to EndGroups method.

Examples:

*-- Generates the report
*
oRep.StartGroups()
DO WHILE NOT EOF()
 oRep.EvalGroups()
 oRep.PrintSection("DEATAIL")
 SKIP
ENDDO
oRep.EndGroups()

See also:


StartGroups, EndGroups, Format files
