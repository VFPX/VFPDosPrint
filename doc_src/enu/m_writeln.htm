WriteLn (Method)

Prints a given text to the current report, and adds a CRLF. Normally, this method is only used by other methods inside the class.

Applies to:


DOSPrint class

Syntax:


Object.WriteLn(cText[,lDoCals][,lNoEject])

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cText Text to be printed.
lDoCalcs Optional. Set it to True if you want to recalculate macro values after text is printed.
lNoEject Optional. Set it to True if you do not want to sends a page break if text overflows to a new page.

See also:


Write, PrintSection
