PrintToText (Method)
(registered version only)

Returns a string with the contents of the current report.

Applies to:


DOSPrint class

Syntax:


Object.PrintToText([,nCopies])

Where:

Object Reference to an object of one of the class mentioned in Applies to:
nCopies Optional. Number of copies to generate. Copies property will be used if this parameter is not indicated.

Examples:

*-- Generate the report to a string
*
oRep.PrintFormat="CUSTOMERS.FMT"
oRep.Run()
cResult=oRep.PrintToText()

See also:


SendToText
