StartConfString (Property)

Sets a string that will be sent to the printer before any other text in the report. This string usually contains control codes to configure the printer.

Applies to:


DOSPrint class

Examples:

*-- Configure the report to use 12 cpi pitch
*
oRep=CREATEOBJECT("cDOSPrint")
oRep.PaperLenght=60
oRep.StartConfString=$C12$
...

See also:


EjectString, NewPageConfString
