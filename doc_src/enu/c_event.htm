Event (Object)

Each event in a format file is represented by an Event object. This objects are automatically created and added to the Events collection when a format file is loaded by setting PrintFormat property. Note that you can change any property of an Event object, but those changes will not be saved to the original format file.

Applies to:


Events collection

Properties
Name Object Source    

Example:

*-- Show some info about INIT event.
*
oDP.PrintFormat="CUSTOMERS.FMT"
oEvent=oDP.Events.Items("INIT")
?oEvent.Name               --> "INIT"
?oEvent.Source             --> "*-- Some code here"
