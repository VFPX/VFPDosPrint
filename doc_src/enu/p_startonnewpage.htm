StartOnNewPage (Property)

Controls wether or not DP will start a new page when a control break occurs. This property is used in group header sections, to allow each group starting in a new page. The default value is False.

Applies to:


Section object

Example:

<groups>
g1 = rectype | g1hdr | g1ftr
</groups>
...
<g1hdr type=g1hdr StartOnNewPage=True>

$BON$RECORD TYPE: %RecType%$BOFF$

</g1hdr>

See also:


ResetPageCounter
