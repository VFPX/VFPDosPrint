GetMacro (Method)

Returns the expression asociated to a previously defined macro.

Applies to:


DOSPrint class

Syntax:


Object.GetMacro(cName)

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cName Name of the defined macro (without $ delimiters) wichs expressions you want to retrieve. If the indicated macro is not defined, method will return an empty string.

Examples:

*-- Get C10 macro expression
*
cC10=oRep.GetMacro("C10")

See also:


AddMacro, SetMacro, Macros
