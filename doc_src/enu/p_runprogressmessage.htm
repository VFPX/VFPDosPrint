RunProgressMessage (Property)

Sets the message showed at the progress window. You can use two special macros in the text asigned to this property:

PAGECNT: Current page count.
CR: Line feed.

Applies to:


DOSPrint class

Example:

*-- Activates progress window with a custom message
*
oDP=CREATEOBJECT("cDOSPrint")
oDP.ShowRunProgress=.T.
oDP.RunProgressMessage="Pages so far: %PAGECNT%"

See also:


ShowRunProgress
