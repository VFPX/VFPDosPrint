SUMMARY Section

Defines the text to be printed at the end of the report. The contents of this section is printed once at the end of the report. If EjectBeforeSummary is setted to True, a page break will be generated before printing this section.

Example:

# CUSTOMERS.FMT
# Basic customer report
#
<config>
StartConfString=$C10$$COFF$    // 10 CPI not condensed (80 cols)
PaperLenght=60
TopMargin=2
LeftMargin=5                 
EjectBeforeSummary=False
</config>

<macros>
CSTATUS=proper(STATUS)
COMPNAME='XYZ Bookstore'
COMPADDRESS='Caracas, Venezuela'
TCOUNT=TRANS(_COUNT[],"9,999")        
</macros>

<header>
#...+....[....+....[....+....1....+....1....+....1....+....1....+....1....+....1
[COMPNAME                         ]                                CUSTOMER LIST
[COMPADDRESS                      ]

CUSTOMER ID           CUSTOMER NAME                          STATUS
====================  ====================================== ==========
</header>

<detail>
[COD               ]  [PROPER(CUSTNAME)                    ] [CSTATUS]
</detail>


<footer>


[datetime           ]                                                   [pageno]
</footer>

<summary>

**************************************
TOTAL CUSTOMER COUNT: [TCOUNT    ]
</summary>
