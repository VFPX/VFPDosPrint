SetMacroValue (Method)

Changes the current value of a defined macro. Note that this method DOES NOT changes macro's associated expression, but its current value.

Applies to:


DOSPrint class

Syntax:


Object.SetMacroValue(cMacro,uValue)

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cMacro Macro to be changed. Note that if the macro does not exists, no error will be generated.
uValue New macro's value.

Examples:

*-- Changes a macro value
*
oRep.SetMacroValue('COMPNAME','Microsoft')

See also:


AddMacro, Macros
