AddMacro (Method)

Adds a new macro to the report macro list.

Applies to:


DOSPrint class

Syntax:


Object.AddMacro(cName,cExpr)

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cName Name of the new macro (without $ delimiters). If there was an existing macro with the same name indicated in cName, no error will be generated and the new macro will be appended at the end of the macro list. However, the first definition of any macro any time it is used inside the report.
cExpr Regular expression wich defines the contents of the macro. This expression will be evaluated while printing each report line, to determinate de most current value of the macro. You can also use constant values such as DATE(), TIME(), "Page:", etc.

Examples:

*-- #1: Constant macro
*
oRep.AddMacro("CUSTNAME","XYZ Bookstore, INC")


*-- #2: Dynamic macro
*
oRep.AddMacro("CUSTNAME","oCustomer.Data.Name")
