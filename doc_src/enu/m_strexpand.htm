STRExpand (Method)

Expands all expressions contained in a given string and returns the result string.

Applies to:


DOSPrint class

Syntax:


Object.STRExpand(cTexto)

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cText Text to expand. Any expression delimited with one of the following delimiters will be expanded:
$$ Evaluates the expression and inserts the result in the string, expanding or reducing it as needed. $NOMEMP$
%% Same as $$ %DATE()%
[] Evaluates the expression and substitutes it with the result, without expanding or reducing the string. [CODIGO  ]


Note that expressions delimited with $$ or %% can change the lenght of the original string, while expressions delimited with [] does not. Also note that expandible expressions can be of any type.

If a expression can not be expanded, it will remains unchanged.

Examples:

LOCAL cNomEmp,cText
cCompany="XYZ Bookstore, INC"
cText=                    "Company: %cCompany%  Date: %date()%"
?oRep.STRExpand(cText) -> Company: XYZ Bookstore, INC  Date: 07-10-2001"
cText=                    "Company: [cCompany]  Date: %date()%"
?oRep.STRExpand(cText) -> Company: XYZ Bookst  Date: 07-10-2001"
