CallSubReport (Method)
(registered version only)

Runs a given format file as a subreport. A subreport is a format file wich is executed and its results returned as a string. This string is then inserted in the calling format file.

Applies to:


DOSPrint class

Syntax:


Object.CallSubReport(cFormatFile)

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cFormatFile Name of the format file to be executed.

Examples:

# Syntax #1: Using CallSubReport method.
<summary>

%DP.CallSubReport('MySummary.FMT')%
</summary>


# Syntax #2: Using special command #SUBREP
<summary>

#SUBREP MySummary.FMT
</summary>
