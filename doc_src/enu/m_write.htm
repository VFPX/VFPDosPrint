Write (Method)

Prints a text in the current report without inserting a line feed.

Applies to:


DOSPrint class

Syntax:


Object.Write(cText[,lLeftMargin])

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cText Texto to print. Note that the method will not add any CRLF to this text, but will process any CRLF contained in the string.
lLeftMargin Optional. If True, the left margin indicated in LeftMargin property will be considered. If False or omited, the text will be printed next to the last text printed.
