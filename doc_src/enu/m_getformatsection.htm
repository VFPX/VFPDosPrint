GetFormatSection (Method)

Returns the contents of any section defined in the current format file. This method applies only if you are using a format file.

Applies to:


DOSPrint class

Syntax:


Object.GetFormatSection(cSeccion)

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cSeccion Name of the secction to retrieve.

See also:


PrintSection, Format files
