DetailString (Property)

Set or gets the string to be printed for each record in the data source. If you are using format files, this property contains the contents of DETAIL section. This property es for read/write.

Applies to:


DOSPrint class

Examples:

*-- Report configuration
*
oRep=CREATEOBJECT("cDOSPrint")
oRep.PaperLenght=60
oRep.HeaderString="CUSTOMER REPORT $CRLF$" + ;
                  "DUE $DATE$"
oRep.DetailString="[CODE] [CUSTNAME              ]"
oRep.FooterString="$DATETIME$                        $PAGENO$"
...

*-- Manual report generation
*
SELECT qCustomers
GO TOP
DO WHILE Not EOF()
 oRep.PrintSection("DETAIL")
 SKIP
ENDDO
..

See also:


TitleString, HeaderString, FooterString, SummaryString
