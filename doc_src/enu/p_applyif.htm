ApplyIf (Property)

Conditional expression to be applicated to the section. This expression is evaluated each time the section is going to be printed. If the conditional fails, the section will not be printed. This lets you to define multiple sections of the same type. You can even have more than one section of any type printed at the same type. If two or more sections are going to be printed at the same time, they will be printed in the order they are declared in the print format.

SPECIAL NOTE: You can use an special runtime variable called DPEOF to define sections that should be printed in the last page of the report. DPEOF will setted to True after the last detail record is readed and printed.

Applies to:


Section object

Example:

# Detail band (Type A records)
#
<detail_a type=detail applyif=(rectype="A")>
[COD] [DESCR_A              ]
</detail_a>

# Detail band (Type B records)
#
<detail_b type=detail applyif=(rectype="B")>
[COD] [DESCR_B              ]
</detail_b>

# Footer band (Normal pages)
#
<footer type=footer applyif=(NOT DPEOF)>

[datetime]                                       [PageNo]
</footer>

# Footer band (Last page)
#
<lastpage type=footer applyif=(DPEOF)>

                  ------ ENF OF REPORT ------

[datetime]                                       [PageNo]
</lastpage>
