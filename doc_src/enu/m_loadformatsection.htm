LoadFormatSection (Method)

Reads a section from the current format file and returns a Section object.

Applies to:


DOSPrint class

Syntax:


Object.LoadFormatSection(cSection)

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cSection Name of the section to retrieve. If there is not such a section inside the current format file, a NULL value will be returned.

Examples:

*-- Reads DETAIL section
*
oDP.PrintFormat="CUSTOMERS.FMT"
oSection=oDP.LoadFormatSection("DETAIL")
?oSection.Text     -> '[COD] [CUSTNAME       ]'
