HeaderString (Property)

Set or gets the string to be printed at the top of every page. If you are using format files, this property contains the contents of HEADER section. This property es for read/write.

Applies to:


DOSPrint class

Examples:

*-- Configuring the report
*
oRep=CREATEOBJECT("cDOSPrint")
oRep.PaperLenght=60
oRep.HeaderString="CUSTOMER REPORT $CRLF$" + ;
                    "DUE $DATE$"
oRep.DetailString="[COD] [CUSTNAME              ]"
oRep.FooterString="$DATETIME$                        $PAGENO$"
...

*-- Generating report
*
SELECT qClientes
GO TOP
DO WHILE Not EOF()
 oRep.PrintSection("DETAIL")
 SKIP
ENDDO
..

See also:


TitleString, DetailString, FooterString, SummaryString
