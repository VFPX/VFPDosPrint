Print (Method)

Send the current report to a specified printer. This method will returns True if the report is successfully sended to the printer or False if any error occurs.

Applies to:


DOSPrint class

Syntax:


Object.Print([cPrinter][,nCopies])

Where:

 

Object Reference to an object of one of the class mentioned in Applies to:
cPrinter Optional. Name of the printer to use. If this argument is not indicated, the default printer will be asumed.
nCopies Optional. Number of copies to generate. Copies property will be used if this parameter is not indicated.

Examples:

*-- Generate the report
*
oRep.PrintFormat="CUSTOMERS.FMT"
oRep.Run()

*-- Send it to a selected printer (2 copies)
*
cPrinter=GETPRINTER()
IF NOT EMPTY(cPrinter)
 oRep.Print(cPrinter,2)
ENDIF

See also:


PrintToFile, SendToPrinter
