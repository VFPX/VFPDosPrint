SetMacro (Method)

Changes the expression asociated with a defined macro.

Applies to:


DOSPrint class

Syntax:


Object.SetMacro(cName,cExpr)

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cName Name of the macro (without $ delimiters) to change. If the macro is not defined, no error will be generated.
cExpr New macro expression.

Examples:

*-- Updates PAGENO macro
*
oRep.SetMacro("PAGENO",nPageNo)

See also:


AddMacro, GetMacro, Macros
