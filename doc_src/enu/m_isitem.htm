IsItem (Method)

Returns True if there is an item in the collection with the indicated name; otherwise, the method returns False.

Applies to:


Collections

Syntax:


Object.IsItem(Name)

Where:

Object Reference to an object of one of the class mentioned in Applies to:
Name Name of the section you are looking for.

Examples:

IF NOT oDP.Sections.IsItem("DETAIL")
 MESSAGEBOX("The format file does not contains a DETAIL section!")
ENDIF
