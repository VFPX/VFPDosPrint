PrintOnNewPage (Property)

Controls wether or not a group header section will be printed again in a new page if details exceeds the current page. This property is only considered on sections used as a group header.

Applies to:


Section object

Example:

<groups>
g1 = rectype | g1hdr | g1ftr
</groups>
...
<g1hdr type=g1hdr PrintOnNewPage=True>

$BON$RECORD TYPE: %RecType%$BOFF$

</g1hdr>
