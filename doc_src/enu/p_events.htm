Events (Collection)

A collection of Event objects. Each item in the collection describes an event section in the current format file.

Applies to:


DOSPrint class

Example:

*-- Show all events in the current format file
*
oDP=CREATEOBJECT("cDOSPrint")
oDP.FormatFile="CUSTOMERS.FMT"
FOR i=1 TO oDP.Events.Count
 ?oDP.Events.Items(i).Name
ENDFOR
