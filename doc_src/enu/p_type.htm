Type (Property)

Determines the type of a section. Section's type is used to determinate if the sections represents:

a) a main section of the format file: HEADER, DETAIL, FOOTER, etc.
b) a section used by a defined control group.
c) an internal or custom event.

For example, when the DETAIL section is to be printed, DP searchs in the Sections collection looking for a section with Type=DETAIL. If more than one section is found, DP uses section's ApplyIf property to determinate wich of them will be printed.
Any section with Type=EVENT will be considered an event and can be called from inside the format using CallEvent method. If a section named Init or Close is found, will be automatically asumed as an event.

Applies to:


Section object

Example:

# This is a detail band. Note that <detail> is equivalent to <detail Type=detail>
#
<detail>
[COD] [CUSTNAME           ]
</detail>

# Also is this
#
<MyDetailBand Type=detail>
[COD] [CUSTNAME           ]
</MyDetailBand>

# This is a custom event
#
<MyEvent Type=event>
*-- Some code here
</MyEvent>

See also:


ApplyIf, Format files
