PrintFormat (Property)

Name of the format file in use. Format files are the easiest way to generate text reports using DOSPrint class.

Applies to:


DOSPrint class

Examples:

*-- Generates CUSTOMER report
*
oREP=CREATEOBJECT("cDOSPrint")
oRep.PrintFormat:="CUSTOMERS01.FMT"
oRep.Run("QCUSTOMERS")
oRep.Print( GETPRINTER() )
