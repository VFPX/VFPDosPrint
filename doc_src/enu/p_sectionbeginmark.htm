SectionBeginMark (Property)

Defines the char or characters used to delimit the beginning of a format section. By default, the value for this property is '<'. SectionBeginMark and SectionEndMark are very usefull when DOSPrint is used to generate formated documents as HTML, XML, etc.

Note that both section mark changes are not applied to CONFIG section, wich allways must be defined using the default section markers (< >).

Applies to:


DOSPrint class

Example:

<config>
SectionBeginMark="[-"
SectionEndMark="-]"
</config>

[-header-]
<html>
<head><title>%HTMLTitle%</title>
</head>
[-/header-]

See also:


SectionEndMark
