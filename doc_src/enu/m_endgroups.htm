EndGroups (Method)

Evalutes all defined groups and generates any required group breaks to finish the current report. This method is only valid if you are using format files. This method is equivalent to EvalGroups method, with lEOF parameter setted to True.

Applies to:


DOSPrint class

Syntax:


Object.EndGroups()

Where:

Object Reference to an object of one of the class mentioned in Applies to:

Examples:
*-- Generates the report
*
oRep.StartGroups()
DO WHILE NOT EOF()
oRep.EvalGroups()
oRep.PrintSection("DEATAIL")
SKIP
ENDDO
oRep.EndGroups()

See also:


StartGroups, EvalGroups, Format files
