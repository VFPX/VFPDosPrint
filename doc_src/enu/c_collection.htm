Collections

A collection is an object with contains a list of related items. Collections are wide used in DP interface.

Properties
Count Items[...]      

 

Methods
IsItem

Example:

*-- Prints text of all sections in a format file
*
oDP=CREATEOBJECT("cDOSPrint")
oDP.FormatFile="CUSTOMERS.FMT"
FOR i=1 TO oDP.Sections.Count
 ?oDP.Sections.Items(i).Text,
ENDFOR
