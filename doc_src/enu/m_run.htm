Run (Method)

Generates a report using the format file indicated in the PrintFormat property.

Applies to:


DOSPrint class

Syntax:


Object.Run([cAlias][,cWhileExpr][,cForExpr])

Where:

Object Reference to an object of one of the class mentioned in Applies to:
cAlias Optional. Name of the table, cursor or view to use in the reporte. If missing, the current alias will be asumed.
cWhileExpr Optional. Conditional expression that is evaluted for each record in the data source. The report processing will stop at the first non-compliant record.
cForExpr Optional. Conditional expression that is evaluted for each record in the data source. If any record does not compliant with this expressión, will be skipped and the report processing will continue until the end of the file.

Examples:

oRep=CREATEOBJECT("cDOSPrint")
oRep.PrintFormat="CUSTOMERS.FMT"
oRep.Clear()
oRep.Run("QCLIENTES","","STATUS=[ACTIVE]")
oRep.Print()
